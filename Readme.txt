A Kolbold Story

A Kolbold Story is an interactive fiction in development for the UCSC class CMPS 148: Interactive Storytelling.
It is the story of an unlucky Kolbold and his loving family.

Developed by: Spencer Apple, Colin Taylor and Derrick Wong

Developed using the tool Inform 7. In order to build it, you will need to download the software. 
A download link can be found here: http://inform7.com/
